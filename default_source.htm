This is the HTML version of the HQDM_FRAMEWORK schema presented in Chapter 17 of Developing High Quality Data Models.

To start, click on the HQDM_FRAMEWORK link.

Select Entities from the next box.

Select a particular entity type name to get the definition for that entity type.

Select the graphic next to the name to see the diagram it appears on.

You can navigate from diagram to diagram using the page connectors.

If you click on an entity type in a diagram you get the definition and EXPRESS for that entity type.

The lexical EXPRESS for the HQDM_FRAMEWORK schema can be found here.
