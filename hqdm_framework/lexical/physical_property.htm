physical_property

A class_of_state that is some characteristic that is the same
for each state that possesses it (is a member_of it).
Example: The color red is a physical_property.

EXPRESS specification:

ENTITY physical_property
SUBTYPE OF ( class_of_state);
SELF\class.member_of  :  OPTIONAL SET [1:?] OF class_of_physical_property;
member_of_kind  :  kind_of_physical_property;
END_ENTITY;

Attribute definitions:

member_of  :  A member_of relationship type where a physical_property may be a member_of one or more class_of_physical_property.
member_of_kind  :  A member_of relationship type where a physical_property is a member_of exactly one kind_of_physical_property.
