sales_product

A class_of_sales_product_instance that is a set of sales_product_instance sold under the same product name.

EXPRESS specification:

ENTITY sales_product
SUBTYPE OF ( class_of_sales_product_instance);
sold_under  :  OPTIONAL SET [1:1] OF product_brand;
meets_specification  :  OPTIONAL SET [1:?] OF requirement_specification;
END_ENTITY;

Attribute definitions:

sold_under  :  A specialization where the sales_product is sold under a product_brand.
meets_specification  :  A subclass_of relationship type where when a sales_product meets_specification of a requirement_specification, each member_of a sales_product is a member_of the requirement_specification.
