kind_of_system

A class_of_system that is also a kind_of_ordinary_physical_
object where each member_of a kind_of_system is of the same
kind.

EXPRESS specification:

ENTITY kind_of_system
SUBTYPE OF ( class_of_system, kind_of_ordinary_physical_object);
has_component_by_class  :  SET [1:?] OF kind_of_system_component;
END_ENTITY;

Attribute definitions:

has_component_by_class  :  A consists_of_by_class relationship type where each member_of a kind_of_system has a member_of one or more kind_of_system_component as a component.
